# Task 3 - Food Ordering Mobile App UI

## Tool Used:

* Figma

## Screens:

* Home Screen
* Food Details Screen

## Features:

* Search Bar
* Food Categories
* Product Cards
* Recently Viewed Section
* Add to Cart Button
* Bottom Navigation Bar

## Description:

A modern Food Ordering Mobile App UI designed using Figma. The application provides a clean and user-friendly interface for browsing food items, viewing product details, and adding items to the cart.
